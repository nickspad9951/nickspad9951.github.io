Table Elements

Tag Name Description
<tr> Table Row The table row element is used to add rows to a table before adding table data and table headings.
<td> Table Data The table data element can be nested inside a table row element, to add a cell of data to a table.
<thead> Table Head The table head element defines the headings of table columns encapsulated in table rows.
<tbody> Table Body The table body element is a semantic element that will contain all table data other than table heading and table footer content.

CSS Attributors

<link> is used to link HTML documents to external resources like CSS files.

Attribute Description
href Specifies the URL to the external resource.
rel Specifies the relationship of the linked document to the current document [Ex. CSS: rel=stylesheet]
type Defines the type of content being linked [Ex. type=text/css]
